Вариантов решения очень много, поэтому подробной проверки не будет (логика в методах, имена переменных и т.п.), мы будем оценивать работу программы по следующим критериям:
- отображение игрового поля при расстановке
- расстановка вертикально
- расстановка горизонтально
- ввод координат не по порядку (для трехпалубного 1,0;3,0;2,0) 
- ввод координат в обратном порядке (для трехпалубного 3,0;2,0;1,0)
- ошибка при вводе чисел отличных от 0-9
- соблюдение ореола
- разные сообщения при разных ошибках
- отсчет оставшегося количества кораблей каждого типа при расстановке
- отображение выстрелов на игровом поле
- сообщения "попал", "мимо", "убил"
- переход хода при промахе
- отсутствие перехода хода при попадании
